**SQL-Data-Warehouse-Analyst-Project** /n
Building a modern data warehouse with Microsoft SQL Server , including ETL  processes,data moddeling and analystics. /n


This project demonstrates a comprehensive data warehousing and analytics solution, from building a data warehouse to generating actionable insights. Designed as a portfolio project, it highlights industry best practices in data engineering and analytics. /n

** Data Architecture** /n

The data architecture for this project follows Medallion Architecture Bronze, Silver, and Gold layers: Data Architecture /n

**📖 Project Overview** /n

**This project involves:** /n

**Data Architecture**: Designing a Modern Data Warehouse Using Medallion Architecture Bronze, Silver, and Gold layers. /n
**ETL Pipelines**: Extracting, transforming, and loading data from source systems into the warehouse. /n
**Data Modeling**: Developing fact and dimension tables optimized for analytical queries. /n
**Analytics & Reporting**: Creating SQL-based reports and dashboards for actionable insights. /n
